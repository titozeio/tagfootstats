# TagFootStats Design System

## Color Usage Rules
- **Background**: Always use `darkBackground` (`#121212`) or `surfaceDark` (`#1E1E1E`).
- **Primary Color**: Use `primaryBlue` (`#005BD6`). 
- **CRITICAL**: Never use deep dark blue (`#013369`) as a primary element background against dark backgrounds. It has poor contrast. Use more vibrant variants like `#005BD6` or `#4285F4`.
- **Accents**: 
  - `nflGold` (`#B3995D`) for primary calls to action, highlights, and important numbers.
  - `accentRed` (`#D50A0A`) for defense and errors.

## Typography
- Use bold, uppercase headers for a "Sporty/NFL" look.
- Use `w900` for important numeric stats.

## Stats Table Guidelines
- Always provide sorting capabilities for numeric columns.
- Use horizontal scrolling for wide tables to maintain readability on mobile.
- Header rows should be distinct (e.g., `Colors.black38`).
